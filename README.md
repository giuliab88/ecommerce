# Archetipo

E-commerce artigianale — ASP.NET Core Web API + Blazor Server (.NET 10)

## Avvio

1. Aprire `ECommerce.slnx` con Visual Studio
2. Tasto destro sulla soluzione → **Imposta progetti di avvio** → **Progetti di avvio multipli** → entrambi su `Avvia`
3. Premere **Start**

Il database viene creato e popolato automaticamente al primo avvio.
